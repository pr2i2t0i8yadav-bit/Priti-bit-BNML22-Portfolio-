# Project Overview

This project is a complete News Portal Website developed as part of the BNML022 assignment activities. The project includes:

1. News Portal Design Prototype  
2. CSS Styling and Visual Identity  
3. CMS-Based News Portal Setup using WordPress and XAMPP  

The portal is designed to provide a modern, responsive, and user-friendly news website experience.

---

# Activities Included

## Activity A1 – Design Prototype
- Homepage wireframe
- Section page wireframe
- Navigation layout
- Featured story area
- Sidebar and footer design

## Activity A2 – CSS Styling
- External CSS stylesheet
- Custom colour scheme
- Styled navigation bar
- Responsive layout
- Uniform news article template

## Activity A3 – WordPress CMS Setup
- WordPress installation using XAMPP
- News categories configuration
- User role management
- Published articles with featured images
- WordPress dashboard and frontend screenshots
